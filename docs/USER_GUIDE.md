# User Guide

## Getting Started

1. Select AI provider (OpenAI or Gemini)
2. Enter research query
3. Click "Start Research"
4. View results and download report

## Sample Queries

- "Find molecules for treating rare respiratory diseases"
- "Identify repurposing opportunities for metformin"
- "Analyze patent landscape for GLP-1 agonists"

## Features

- Multi-agent research
- Real-time data fetching
- Automated report generation
- Cost tracking
